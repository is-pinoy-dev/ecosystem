Introduce badge-kit — a Cloudflare Worker that generates SVG/PNG/WebP badges and banners for is-pinoy.dev subdomains. Includes 8 badge types, pixel sun design, IBM Plex Mono eyebrows, CRT scanline banner, registry subdomain verification, and WASM-based raster rendering. Docs site adds badge-kit reference and preview components.
